# CoupleConnect

CoupleConnect — Android-приложение для отслеживания статистики ваших отношений. Приложение помогает фиксировать дату начала отношений, важные события и рассчитывать общее время, проведенное вместе. 

## Функционал

- Указать своё имя и имя партнёра. 

- Задать дату начала отношений и загрузить фотографии. 

- Просматривать общее количество дней, недель, месяцев и лет вместе. 

- Отслеживать события («моменты») — записи с датой, временем, местоположением и описанием. 

- Просматривать статистику событий за разные периоды (неделя, месяц, квартал, год). 

## Особенности

- Удобный интерфейс для просмотра истории отношений и событий. 

- Сохранение и отображение личных данных пары. 

- Статистика ключевых событий за выбранный период.

## Инструменты и библиотеки

Проект написан на языке Kotlin и использует стандартный Android SDK для реализации пользовательского интерфейса и логики приложения. В качестве базы данных была использована SQLite, а в качестве ORM - Room. 

## Установка

1. Cклонируйте репозиторий:
```bash
git clone https://github.com/icestormikk/CoupleConnect.git
```
2. Откройте проект в Android Studio.

3. Соберите и запустите на эмуляторе или реальном устройстве.
